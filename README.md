# ⚡ Física Aplicada

Aplicación interactiva desarrollada en Python con Streamlit para resolver problemas de:

- 🔹 Electrostática (Ley de Coulomb)
- 🔹 Circuitos eléctricos simples
- 🔹 Producto vectorial y fuerza magnética

La aplicación muestra cálculos automáticos, gráficas interactivas y procedimientos matemáticos paso a paso con enfoque educativo.

---

# 🚀 Características principales

## ⚡ 1. Ley de Coulomb
- Cálculo de fuerza eléctrica neta
- Soporte para múltiples cargas
- Visualización tipo GeoGebra
- Desarrollo matemático paso a paso
- Interpretación física del resultado

---

## 🔌 2. Circuitos simples
Soporte para:

### Resistores
- Serie
- Paralelo

### Capacitores
- Serie
- Paralelo

La aplicación calcula:
- Resistencia equivalente
- Capacitancia equivalente
- Corriente total
- Voltajes
- Cargas
- Resumen por componente
- Verificación de resultados

Además:
- Dibuja automáticamente el circuito
- Explica el procedimiento completo
- Soporta prefijos eléctricos

---

## 🧲 3. Producto vectorial y fuerza magnética

### Producto vectorial
La aplicación:
- Calcula el producto vectorial de dos vectores
- Muestra el desarrollo usando determinantes
- Grafica los vectores en 3D

La aplicación:
- Calcula la fuerza magnética
- Obtiene el vector del conductor
- Aplica producto vectorial
- Muestra interpretación física
- Visualiza vectores en 3D
